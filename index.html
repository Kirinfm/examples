Basic Examples

Learn the fundamentals of the HERE Maps API for JavaScript through these basic examples. Source code snippets are highlighted and the HTML+CSS+JavaScript content can be easily viewed on Github. To modify and build on existing examples, fork the 'examples' repository and start experimenting with the HERE Maps API.

Adding a marker to the map

Zooming to all markers
